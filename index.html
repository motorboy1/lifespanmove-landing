왜 LifespanMove 인가

연 1회 측정실에서, 매월 일상에서로

기존 체력 측정은 연 1회 측정실 방문이 필요하고 등급만 제공합니다. LifespanMove는 일상 환경에서 자동으로, 정량 데이터와 또래 비교까지 제공합니다.

기존 측정 (PAPS · 국민체력100)

  • 연 1회 측정
  • 측정실 방문 필요
  • 등급만 제공
  • 학부모 공유 수단 없음

LifespanMove

  • 매월 무자각 자동 측정 (8초 룰)
  • 일상 환경에서 측정
  • 정량 점수 + 5개 세부 점수 + 또래 비교
  • 학부모 자동 리포트 (PDF)

요금

시설 규모에 맞는 도입

시설 SaaS는 시설당 월 구독으로 회원 전원의 측정·리포트·통계를 제공합니다. 개인용 5단계 요금도 함께 제공합니다.

투자·파트너

모방하기 어려운 데이터 자산

단일 PI가 단일 프로토콜로 확보한 전 생애 cohort는 글로벌 희소성을 가집니다. 저출산과 고령화 양극단을 동시에 헤지하는 시장 구조입니다.

IP

데이터 IP 희소성

유치원~80대 450명을 단일 프로토콜로 측정. 모방에 5~10년과 수억원이 필요한 진입 장벽.

↔

인구동학 양극단 헤지

저출산(1인당 지출 증가) + 고령화(시장 확대)를 동시에 포착하는 수요 구조.

4M

4-modality 로드맵

IMU + Markerless + Force plate + Smartphone 융합으로 정밀도와 접근성을 동시에 확보.

3년 ARR 목표 14~22억원 (보수 시뮬: 도장 500 시설 + B2C 1,000~3,000명). 영남대 산학협력 기반.
